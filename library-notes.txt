#       {"cmd":"led","which":"green|amber|off","on":true|false}
#   - Fire action is hard-gated by 'armed' state and issues a HIGH pulse on PIN_PULSE_OUT for PULSE_WIDTH_MS.
#   - Status bi-colour LED is mutually exclusive via setStatusLED().
